    This is a motherboard, it is the center of all computers. It contains a spot for many things such as the CPU and RAM. The motherboard as a whole allows the computer components to talk to each other. A motherboard is created by printing out a board made of glass and fibers. The components are put onto the motherboard and are linked together with copper wires in the board. For the computer to work. you need to add a CPU and RAM for it to run and to run faster.
