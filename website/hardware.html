    • Hardware That I Used

    1.

    This is a motherboard, it is the center of all computers. It contains a spot for many things such as the CPU and RAM. The motherboard as a whole allows the computer components to talk to each other. A motherboard is created by printing out a board made of glass and fibers. The components are put onto the motherboard and are linked together with copper wires in the board. For the computer to work. you need to add a CPU and RAM for it to run and to run faster.

    2.

    The next part I will talk about is the CPU. Otherwise known as the Central Processing Unit. The CPU processes all the information and takes in an input from differen parts of the computer and outputs information or does the action. It takes inputs from the RAM (Random Access Memory) and "executes" it and displays something. The number of hertz on the CPU, is how many intructions it can do in a second. Such as, if it was 3.9Hz (Hertz), it can process 3.9 billion processes per second.


    3.

    Next, there is the GPU, which stands for the Graphics Processing Unit. It improves the graphics and videos. It's used for computing 3D functions. If there wasn't a dedicated graphics card, the CPU would perform slower because it has to also have to display something on the screen. The GPU will make the computer more efficient and much more faster by letting a seperate unit do the graphics.


    4.

    Finally, there is the PSU, which stands for Power Supply Unit. This powers all the units in the computer and keeps the computer running. It converts the main AC power from an outlet into low-colage regulated DC power. PSU's come in differnt types such as form the size and how much voltage tolerances.
